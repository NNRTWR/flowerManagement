## How to run tailwind css
first time or have conflict
```sh
npm run build
```
 ```sh
npm run dev 
 
```
**ต้องโหลด node.js ก่อนนะ ยังไม่ได้ทำใส่ docker**
สามารถรันปกติได้ ไม่จำเป็นต้องรันอันนี้ถ้าจะเทสค่า
